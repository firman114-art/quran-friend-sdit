🎨 Konversi Icon AISHA ke PNG

Generator PNG untuk PWA (192x192 dan 512x512)

Icon SVG (Source)

📋 Cara Menggunakan

  1. Klik tombol "Download 512x512 PNG" di bawah
  2. Simpan sebagai icon-512x512.png
  3. Ulangi untuk ukuran 192x192
  4. Pindahkan kedua file ke folder public/
  5. Rebuild aplikasi: npm run build


📥 Download SVG Source

💡 Tips

  • File SVG bisa langsung dibuka di browser Chrome/Firefox
  • Gunakan tool online seperti CloudConvert untuk konversi otomatis
  • Atau gunakan Figma/Illustrator untuk export dengan kualitas terbaik

📁 Struktur File Target

public/
├── manifest.json
├── service-worker.js
├── icon-aisha-simple.svg    ← Source
├── icon-192x192.png      ← Hasil konversi
└── icon-512x512.png      ← Hasil konversi
